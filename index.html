1 ¿Como empezar? 🤔

¡Hola! Para este curso vamos a usar 2 sitios: Github y P5.js.

Github va a ser algo así como nuestro salón de clase, o nuestra biblioteca. Ahí estará el material y los enlaces que necesitaremos.

Con P5.js vamos a hacer pequeños programas y a visualizarlos. Estos programas son llamados “sketches” (o “bocetos”).

1.1 Github

  1. Entra a https://github.com y haz una cuenta.

❗️ Tu cuenta será visible a todo el mundo, así que de preferencia usa un pseudónimo corto y formal para tu nombre de usuario (lo puedes cambiar después, pero con algunas complicaciones).

1.2 P5.js

  1. Entra a su página principal: https://p5js.org/ (puedes cambiar el idioma de la página en la esquina ↗︎).
  2. Entra al editor (o desde aquí: https://editor.p5js.org/)

☝️ No es indispensable hacer una cuenta para usar el editor, pero la necesitaras para guardar y compartir tus programas.

Para hacer una cuenta en P5.js:

  1. Ya dentro del editor, haz click en sign up en la esquina ↗︎.
  2. Usa tu cuenta de github para ingresar (recomendado) o llena los datos.

❗️ Ten en cuenta que el nombre que hayas elegido será visible en todos tus proyectos (de preferencia usa un pseudónimo).

¡Eso es todo! ¡Ya estamos listos para comenzar a programar!
